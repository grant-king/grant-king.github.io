Game Bot

Custom Python program controls Forza Horizon 5
Bot repeatedly searches and scans a list of live player auctions, then saves each when they are complete
pynput library automates keyboard and mouse control
Custom methods use random gaussian timing to emulate human input
OpenCV library for template matching enables intelligent, robust navigation within the game

OCR Routines

Optical Character Recognition implemented with OpenCV Python library
Areas of interest are captured as an image and translated to text before  the data is stored
Semi-transparent text over dynamic backgrounds in the game make OCR challenging for this application
Each screen area requires a unique combination of image thresholding and filtering settings for the OCR to be accurate
Custom OCR optimization tool determines the best thresholding settings for each sample area

Data Collection

Using Python pandas library
Stores all raw data for analysis
Each record contains 14 regular attributes: information about the car model, the individual car, and the auction
Any entries containing incomplete data are flagged; this constitutes between 1% and 2% of all entries
Data cleaning routine analyses raw data, fixes what it can, and discards bad data 

Data Analysis

Clean data is stored in a new file
This data is then analyzed, organized, and sorted by completion date and time
About half of all auctions complete without successful bids, these are stored but not used yet
Successful auctions are organized into subsets by car manufacturer; these are used for visualization

Data Visualization

Using Python Bokeh library
Interactive scatter plots are  procedurally generated for each manufacturer
123 plots total, representing more than 600 different car models
Plots display winning bid amount and the time the auction was saved on the axes
Interactive plots let users scale the graph and zoom in on subsets of the data
Users can show or hide car models by clicking on the legend
Plot markers show the car's individual performance rating by variable marker sizes
Plot markers are semi-transparent so they can overlap without obscuring each other at different zoom levels
Custom colorizing routine picks a random, uniquely distinguishable, and highly visible color for the markers for each car model

Presentation Website

Static GitHub Pages site has free hosting, only a $12 annual maintenance cost for the domain registration
Python script effortlessly generates the plots to update the website with fresh data every day
Responsive layout with Bootstrap 5
Traffic source is exclusively free Google searches, analytics shows nearly 2K unique users this month
